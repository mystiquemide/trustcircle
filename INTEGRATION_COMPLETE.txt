================================================================================
                            QUICK START GUIDE
================================================================================

DEVELOPMENT (Backend + Frontend)
  1. Start Backend:
     $ cd backend && npm start
     Expected: Server on http://localhost:5000

  2. Start Frontend:
     $ cd frontend && npm run dev
     Expected: App on http://localhost:5173

  3. Test Flow:
     - Create a circle → Check API call in Network tab
     - Join a circle → Check API POST to /api/invites/*/joined
     - View dashboard → Check circle metadata sync

OFFLINE MODE (Blockchain Only)
  1. Don't start backend (or stop it)
  2. Start frontend: $ cd frontend && npm run dev
  3. Create/join circles → Works normally
  4. Check console → API errors logged (non-blocking)

PRODUCTION DEPLOYMENT
  1. Update VITE_API_BASE_URL to production backend URL
  2. Build: $ npm run build
  3. Deploy frontend to CDN/hosting
  4. Deploy backend API server
  5. Test integration with production backends
